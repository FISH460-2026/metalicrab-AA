## A Quarto Manuscript Template
Testing pollution stressors on immunology response in hairy shore crabs by exposing them to heavy metals commonly found in polluted shipping ports.
Materials Needed: 14 shore crabs, 3 small box tanks, ~20 needles for hemolymph extractions, 4 microscopes, 4 hemocytometers & coverslips, lead weight and copper weight of the same mass, scale, ruler, count clicker, blue staining for hemocyte identification
